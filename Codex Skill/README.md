# Codex Skills Repository (Prototype)

This folder contains specialized skills designed for **Codex** to use within the **67-multi-agent-workflow** framework.

> [!IMPORTANT]
> This is a **prototype** environment. These skills are currently optimized and wired specifically to work within this repository. 
> 
> **Future Roadmap**: We plan to update these skills to be framework-agnostic so they can be used in any environment in the future.

## Current Skills
- **dart-simplifier**: Automated logic for cleaning and simplifying Dart/Flutter code.
